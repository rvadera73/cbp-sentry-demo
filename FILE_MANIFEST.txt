QUICK FILE REFERENCE
────────────────────

To start development:
  → Read: README.md or CLAUDE.md
  → Run: docker-compose up
  → Visit: http://localhost:3000 (UI)
  → Visit: http://localhost:8000/health (API health)

To implement a service:
  → Edit: api/services/{service}/routes.py
  → Create: api/services/{service}/logic.py (new)
  → Test: api/tests/test_{service}.py

To update styling:
  → Edit: ui/tailwind.config.ts (colors, fonts)
  → Edit: ui/src/index.css (global styles)

To add a page:
  → Create: ui/src/pages/YourPage.tsx
  → Update: ui/src/App.tsx (add route)
  → Update: ui/src/components/layout/DemoStepper.tsx (if workflow step)

To deploy:
  → GCP: gcloud builds submit --config=deploy/gcp/cloudbuild.yaml
  → AWS: git push + CodePipeline (configured with buildspec.yml)
