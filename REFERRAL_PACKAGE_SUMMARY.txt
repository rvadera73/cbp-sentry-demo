================================================================================
                            USAGE IN APPLICATION
================================================================================

In CaseSplitPane.tsx:
  {selectedCase.risk_score >= 40 && (
    <ReferralPackage
      data={buildReferralPackageData(selectedCase)}
      onExecuteReferral={(notes) => {
        // Handle TRLED referral submission
      }}
      onHoldExamine={(notes) => {
        // Handle hold-for-examination submission
      }}
      onOverride={(justifications, notes) => {
        // Handle override decision with officer justifications
      }}
    />
  )}

The buildReferralPackageData() helper transforms CaseCardData to ReferralPackageData:
  - Distributes risk across H1 (40%), H2 (35%), H3 (25%) vectors
  - Generates sample discrepancies and entity chain
  - Creates conditional scenarios for what-if analysis
  - Maps existing case fields to referral structure
